\doxysubsection{Detailed Description}
This is a wait instruction similar to wait instruction on industrial controllers. 

The instruction has several modes of operation.


\begin{DoxyItemize}
\item TIME \+: This will wait for a specified number of seconds and then continue
\item DIGITAL\+\_\+\+INPUT\+\_\+\+HIGH \+: This will wait for a digital input to go high(1) then continue
\item DIGITAL\+\_\+\+INPUT\+\_\+\+LOW \+: This will wait for a digital input to go low(0) then continue
\item DIGITAL\+\_\+\+OUTPUT\+\_\+\+HIGH \+: This will wait for a digital output to go high(1) then continue
\item DIGITAL\+\_\+\+OUTPUT\+\_\+\+LOW \+: This will wait for a digital output to go low(0) then continue 
\end{DoxyItemize}
