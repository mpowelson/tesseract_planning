\doxysubsection{Detailed Description}
This struct is passed as an input to each process in the decision tree. 

Note that it does not have ownership of any of its members (except the pointer). This means that if a \mbox{\hyperlink{structtesseract__planning_1_1TaskInput}{Task\+Input}} spawns a child that is a subset, it does not have to remain in scope as the references will still be valid
