\doxysubsection{Detailed Description}
This instruction indicates that a timer should be started and when the time expires it either sets a digital output high(1) or low(0). 


\begin{DoxyItemize}
\item DIGITAL\+\_\+\+OUTPUT\+\_\+\+HIGH \+: The digital output will be set to high(1) when the timer expires
\item DIGITAL\+\_\+\+OUTPUT\+\_\+\+LOW \+: The digital output will be set to low(0) when the timer expires 
\end{DoxyItemize}
